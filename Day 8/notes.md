    Converting custom objects into dictionaries.
    Reconstructing objects from dictionaries.

💡 Key Takeaways
    JSON is one of the most common formats for storing and exchanging structured data.
    Custom Python objects cannot be stored directly in JSON; they must first be converted into dictionaries.
    Saving application data to JSON allows information to persist even after the program closes.
    Using to_dict() and from_dict() methods makes object serialization clean and reusable.
